# NLP Assignment

This repository contains my Natural Language Processing (NLP) laboratory assignments implemented using **Python**, **NLTK**, **Scikit-learn**, and **Jupyter Notebook**.

## 📚 Objectives

The assignment demonstrates fundamental NLP concepts and text preprocessing techniques using a custom dataset (`foodie.txt`).

## 📝 Exercises

- Exercise 1 – Reading a Text File
- Exercise 2 – Sentence and Word Tokenization
- Exercise 3 – Text Lowercasing
- Exercise 4 – Punctuation Removal
- Exercise 5 – Word Frequency Distribution
- Exercise 6 – Stop Word Removal
- Exercise 7 – Stemming using Porter Stemmer
- Exercise 8 – Frequency Distribution after Stemming
- Exercise 9 – TF-IDF, Cosine Similarity, Document Ranking, and PCA Visualization
